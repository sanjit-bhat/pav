specs:
- KT ptr: (γ, ep, uid) ↪KT opt_pk.
- Get: γcli ↪KT.
- SelfMon: is_staged γcli start_ep keys.
  + start_ep is when Client first started SelfMon'ing its key.
  i.e., the first epoch of keys.
- Audit: is_audit γcli γadtr end_ep.
  + end_ep is epoch of Audit.
  + impl:
    (1) first cli digs = auditor digs, up thru end_ep.
    (2) mono_plain from γadtr.(start) to end_ep.

lemmas:
- kt_ptsto_txfer (Get + Audit):
(γcli, ep) ↪KT → is_audit γcli γadtr end →
γadtr.(start) ≤ ep ≤ end → (γadtr, ep) ↪KT.
- commit_staged (SelfMon + Audit):
is_staged γcli start_keys keys →
is_audit γcli γadtr end →
γadtr.(start) ≤ start_keys →
start_keys + length keys ≤ end →
(∀ ep key, keys !! ep = Some pk → (γadtr, ep) ↪KT pk)
- combine_audits (prove later):
is_audit γcli γadtr0 start0 end0 →
is_audit γcli γadtr1 start1 end1 →
(start0, end0) "overlaps" (start1, end1) →
is_audit γcli γadtr0 (start0 `min` start1) (end0 `max` end1).
